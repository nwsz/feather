Introduction

Feather docs

A whisper-light Chrome extension that watches Discord for new messages and logs them with webhook alerts and Supabase storage.

Live Detection MutationObserver watches Discord in real time.
Webhook Alerts Send clean Discord notifications instantly.
Supabase Logging Permanent searchable message storage.
Smart Suppression Blocks false triggers when switching channels.

Quick Start

1. Load unpacked extension
2. Open Discord
3. Add webhook
4. Add Supabase
5. Start logging
Getting Started

Installation

Load Feather manually through Chrome extensions. 

chrome://extensions

Enable Developer Mode
Click Load Unpacked
Select feather folder

Supabase

Insert every detected message into a database row. 

create table feather_logs (
 id bigint generated always as identity primary key,
 author text,
 content text,
 channel text,
 count integer,
 timestamp timestamptz default now()
);

Reference

Detection Logic

Feather uses DOM detection with deduplication. 

1. Find new message node
2. Ignore history burst inserts
3. Ignore duplicates
4. Extract author/channel/content
5. Trigger integrations
